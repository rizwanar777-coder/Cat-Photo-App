# Cat-Photo-App
This is a basic HTML project built The Cat Photo App displays cat images, lists of things cats love and hate, and includes a simple form.
 Technologies Used
- HTML
 Features
  1.Displays cat images
  2.Shows lists of things cats love and hate
  3.Includes a form with input fields and buttons
  4. Uses basic HTML elements like headings, lists, and forms
 What I Learned
- Basic HTML structure
- Using headings, paragraphs, and images
- Creating ordered and unordered lists
- Building simple forms with input and buttons
